## Requirements

- A C++11 (or higher) compiler
- CMake

## Setup (Unix-like)

Install Raylib ([Linux](https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux), [macOS](https://github.com/raysan5/raylib/wiki/Working-on-macOS)):

macOS:
```bash
brew install raylib
```

Arch Linux:
```bash
sudo pacman -S raylib
```

Compile via CMake:

```bash
mkdir build
cd build
cmake ..
```
